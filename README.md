# .config

My personal configuration for all the tools and random shit I use on a daily basis:

- [ghostty](https://ghostty.org/)
- [neovim](https://www.lazyvim.org/)
- [tmux](https://github.com/tmux/tmux/wiki)
- [zsh w/ omz](https://ohmyz.sh/)

## Getting started

Clone this repository to your XDG base directory (usually `~/.config`)

```bash
mv ~/.config ~/.config.bak
git clone https://github.com/JoeyMckenzie/.config.git ~/.config
```

I use a handful of plugins that can be installed with `bin/init.sh`
